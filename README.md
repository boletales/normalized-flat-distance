# 「平地換算距離」 ― 運動負荷の数理的考察に基づくサイクリングコースの加法的難易度指標の提案
## 序論
登山コースには加法的な難易度指標がある。ハイキング情報共有サイト YAMAP は「コース定数」という加法的な指標を表示し、「合計12未満ならやさしい、25以上ならきつめ」と目安を示して、計画作成の一助としている。

サイクリングコースの難易度も距離や登った高さの合計（獲得標高）で間接的に表されるが、これらを一次元の指標にまとめられるだろうか。たとえば、次の四つのコースを考える。

- 平坦$`100\,\mathrm{km}`$
- $`2\%`$の登り$`10\,\mathrm{km}`$
- $`4\%`$の登り$`5\,\mathrm{km}`$
- $`20\%`$の登り$`1\,\mathrm{km}`$

獲得標高を見れば$`2\%`$ $`10\,\mathrm{km}`$、$`4\%`$ $`5\,\mathrm{km}`$、$`20\%`$ $`1\,\mathrm{km}`$はすべて同じ$`200\,\mathrm{m}`$ upであるが、$`4\%`$ $`5\,\mathrm{km}`$は素人に登りきれても$`20\%`$ $`1\,\mathrm{km}`$は困難である（目安として、コンクリ舗装で丸い模様のある坂は$`15\%`$程度である）。また、$`2\%`$ $`10\,\mathrm{km}`$よりは平坦$`100\,\mathrm{km}`$のほうが大変だが、距離と高さを統一的に扱う手段なしにこれらの比較はできない。

ところで、スポーツ科学では、自転車を走らせるために投入する仕事率「パワー」を測定して瞬間的な運動負荷を定量的に扱う方法が普及している。走行に必要なパワーは速度と勾配から決まり、たとえば「平坦路を$`31\,\mathrm{km/h}`$で走るのと、$`4\%`$の坂を$`19\,\mathrm{km/h}`$で走るのはいずれも$`200\,\mathrm{W}`$のパワーが必要である」というように、平坦路と斜路で同一の基準を用いた議論ができる。与えられたコースに対し、各点での予想走行速度を定めれば、パワーを計算し、コース全体の運動負荷を定量的に評価することができるだろう。

評価にあたって、速度-パワーの関係が斜度依存的であることに留意せねばならない。次節で触れる通り、平坦路においては空気抵抗に起因する速度の三乗の項が、急坂路では位置エネルギー変化に起因する速度の一次の項が支配的である。すなわち、登りにパワーを投資すれば線形的に早くなるのに、平地でパワーを投資しても三乗根でしか早くならないため、賢いペーシング戦略が必要となる。

本稿では、サイクリングコースを走破する「つらさ」をスポーツ科学的側面から考察して定義し、おなじつらさで早く走れる「賢いペーシング」を最適化問題として定式化する。この問題の解からは、サイクルスポーツで広く知られている事実が再発見される：

- 初心者ほど登りをつらく感じる。
- 平坦路ではほどほどに手を抜き、登りでは頑張ったほうがよい。
- まともに登れる坂であれば、獲得標高はつらさのよい指標である。

さらに、最適なペースはコース・力量によらず「平地をどの速度で走るか」のみによって決まり、急坂ではいつでもだれでも平地の1.35倍程度のパワーで走るべきであるという驚くべき結論を得た。

そののち、得られたペーシングに現実的な修正を加えて、サイクリングコースの大変さを表す新しい指標「平地換算距離」を提案する。

## 物理的・生理学的前提

### パワーのモデル

サイクリングの客観的な運動負荷は、サイクリストがクランクに対して発揮する仕事率（「パワー」と呼ばれる）によって測定される。無風の条件下で角度 $`\theta`$ の道路を速度 $`v`$ で定速走行する場合のパワー $`P_{\mathrm{precise}}(\theta, v)`$ は、次の式で表される：

```math
P_{\mathrm{precise}}(\theta, v) = \frac{1}{\eta}(\frac{1}{2}\rho C_dA\,v^3 + mg(\sin\theta + C_{rr}\cos\theta)\,v + m v\frac{dv}{dt})
```

ここで、ρは空気の密度、Cdは空気抵抗係数、Aは前面投影面積、mは総質量、gは重力加速度、Crrは転がり抵抗係数、ηは駆動効率である。これらの係数は、サイクリングの一般的な条件下ではおおむね定数であるとみなすことができる。
これらの大まかな値を示すと、気温20℃、海面上の空気の密度はおよそ$`1.2\,\mathrm{kg/m^3}`$である。ロードバイクの場合、$`C_dA`$はおよそ$`0.3`$〜$`0.5\,\mathrm{m^2}`$、$`C_{rr}`$はおよそ$`0.005`$、$`\eta`$は$`95`$〜$`98\%`$である。また、車重は$`6`$〜$`10\,\mathrm{kg}`$、サイクリストの体重は$`50`$〜$`100\,\mathrm{kg}`$程度であり、$`100`$〜$`300\,\mathrm{W}`$程度のパワーを継続して発揮できる。

今回は指標を単純にするため、加減速に由来するパワーを無視する（サイクリストがクランクにする仕事のほとんどは位置エネルギーを得たり風を起こしたりするために空費され、運動エネルギーのやりとりは大きくない。全質量$`70\,\mathrm{kg}`$、速度$`10\,\mathrm{m/s}`$のときの運動エネルギーはたったの$`3500\,\mathrm{J}`$であるが、これは位置エネルギー$`5\,\mathrm{m}`$分にしかならないし、この速度域での空気抵抗の仕事率は$`250\,\mathrm{W}`$に達する）。
また、上記の係数は、一回のサイクリングで大きく変わることはないから定数であるとみなすことにする。これ以降、係数を $`a \in \mathbb{R}`$ と、$`b : [-\pi/2, \pi/2] \to \mathbb{R}`$ つまり、

```math
a := \frac{\rho C_dA}{2\eta}, \qquad b(\theta) := \frac{1}{\eta}mg(\sin\theta + C_{rr}\cos\theta)
```

にまとめて、$`P_{\mathrm{precise}}`$ の近似値

```math
P_{\mathrm{approx}}(\theta, v) := av^3 + b(\theta)v
```

を考えることにする。これは速度を均衡させるために必要なパワーであるから、「均衡パワー」と呼ぶことにする。

### 主観的な運動負荷のモデル

Andrew Coggan らは、サイクリストの発揮したパワーと筋肉中の乳酸濃度(後者がおよそ前者の4乗に比例するらしい)の関係から、Normalized Power® (NP)という指標を提案し、Stravaを始めとしたアクティビティ記録サービスで採用されている。NPは、その回の運動で発揮したパワーを30秒移動平均で平滑化したのちL4ノルムを取ったものである。サイクリングを頑張った際のNPはサイクリストの力量によっておおまかに決まることが知られている。

NPの四乗は加法的な指標となる。ここでは、コースをある定められた速度分布のもと走った際の均衡パワーの四乗の時間積分（「合計つらさ」と呼ぶことにする）をサイクリストの主観的な運動負荷の指標とする。

## 数理的考察

### コースとつらさの定義

**Def. 1** 正実数 $`L`$ 、関数 $`G: [0, L] \to [-\pi/2, \pi/2]`$ の組 $`(L, G)`$ を「コース」と呼ぶ。$`L`$ を「コース長」と呼び、$`x \in [0, L]`$ に対して $`G(x)`$ を「$`x`$ における勾配」と呼ぶ。

**Def. 2** 正実数 $`a`$、関数 $`b: [-\pi/2, \pi/2] \to \mathbb{R}`$ の組 $`(a, b)`$ を「抵抗モデル」と呼ぶ。$`a`$ を「空気抵抗係数」と呼び、$`\theta \in [-\pi/2, \pi/2]`$ に対して $`b(\theta)`$ を「重力抵抗係数」と呼ぶ。正実数 $`v`$ と $`\theta \in [-\pi/2, \pi/2]`$ に対して $`av^3 + b(\theta)v`$ を「勾配 $`\theta`$、速度 $`v`$ での均衡パワー」と呼ぶ。

**Def. 3** コース $`C (= (L, G))`$、抵抗モデル $`M`$、正実数 $`T`$、微分可能で単調増加な全単射 $`D: [0, T] \to [0, L]`$ で、$`D(0)=0`$ かつ $`D(T)=L`$ であるものの組 $`(C, M, T, D)`$ を「走行」（もしくは「走行の時間表示」）と呼ぶ。$`T`$ を「走行時間」と呼び、$`t \in [0, T]`$ に対して $`D(t)`$ を「$`t`$ における位置」と呼ぶ。

**Def. 4** 走行 $`R (= ((L, G), M, T, D))`$ と $`t \in [0, T]`$ に対して、$`R`$ の $`t`$ における

- 「瞬間速度」を、$`v := dD/dt(t)`$ と定める。
- 「瞬間均衡パワー」を、抵抗モデル $`M`$ の勾配 $`G(D(t))`$、速度 $`v(t)`$ での均衡パワーで定める。
- 「瞬間つらさ」を、$`S := P^4`$ と定める。

**Def. 5** 走行 $`R (= (C, M, T, D))`$ に対して「合計つらさ」を、$`R`$ の $`t`$ における瞬間つらさを $`S(t)`$ として、$`\int_0^T S(t)\,dt`$ と定める。

**Def. 6** 走行 $`R`$ に対して、$`R`$ の合計つらさ/$`R`$ の走行時間 を「平均つらさ」と呼ぶ。（これはつまり、NPのようなものである）

- 「地点速度」を、$`v := t`$ における瞬間速度で定める。

**Def. 10** 抵抗モデルが同じ走行(速度表示) $`R_1 (= (C_1, M, v_1))`$、$`R_2 (= (C_2, M, v_2))`$ について、「$`R_1`$ と $`R_2`$ の結合」とは、走行 $`R_3 (= (C_3, M, v_3))`$ である。ここで、$`C_3`$ は $`C_1`$ と $`C_2`$ の結合であり、$`v_3(x) := x < L_1`$ なら $`v_1(x)`$、さもなくば $`v_2(x - L_1)`$ とする。定義より明らかだが、結合の走行時間は $`R_1`$ と $`R_2`$ の走行時間の和である。

**Prop. 3** 合計つらさは加法的である。すなわち、抵抗モデルが同じ走行 $`R_1, R_2`$ に対して、$`R_1`$ と $`R_2`$ の結合の合計つらさは、これらの合計つらさの和である（合計つらさの定義より直ちに明らか）

### 走行の最適化

#### 概略

サイクリストが達成できるNPは力量によって上限が定まる。サイクリングにおいて、定められたNP以下で最も早くコースを走り切る走行を考えるのは自然な問題設定である。（同じタイムで最も低いNP」だと考えても、同種の問題に帰着する）

本節では、前節で定義した「平均つらさ」を一定に保ちつつ、走行時間を最小化する最適な走行がどのようものであるか考えることにする。定性的には、次のような結論が得られる。

- 各地点で発揮すべきパワーは、その地点の勾配のみによって定まる。
- 平地ではパワーを少なめに、登りではパワーを多めに発揮するべきである。（平地で飛ばしても空気抵抗が増えるばかりで割に合わないため、このような配分になる。この結果は我々の実感と符合する。）
- 勾配-つらさのプロットは登り勾配側では一次関数に漸近する。
- 急坂路でのパワーは抵抗モデルによらず平地の1.3〜1.38倍程度になる。
- 驚くべきことに、抵抗モデルと特定の勾配での速度をひとつ定めると、どのようなコースであっても同じペーシングが最適となる。

特に、パワーに関する諸結論はサイクリングの一般的アドバイスと整合的であるから、次節が与える結論は荒唐無稽ではないだろう。

#### 定式化

特定の抵抗モデル $`M (= (a, b))`$ にとって各コースで最適である走行がどのようなものか調べてみよう。

仮の問題設定として、サイクリストの力量を表す正実数 $`S_0`$ とコース $`C (= (L, G))`$ が与えられたとき、$`v: [0, L] \to \mathbb{R}_+`$ であって、走行 $`(C, M, v)`$ の平均つらさが $`S_0`$ であり、走行時間が最小となるものを考えることにする。

この走行の合計つらさは、各点 $`x`$ における地点速度を $`v(x)`$ として $`\int_0^L (a v(x)^3 + b(G(x)) v(x))^4 dx`$ である。また、走行時間は $`\int_0^L 1/v(x)\,dx`$ である。これらを用いて、平均つらさが $`S_0`$ であることとは、次の等式が満たされることである。

```math
\left(\int_0^L \frac{(a v(x)^3 + b(G(x))v(x))^4}{v(x)}\,dx\right) / \left(\int_0^L \frac{1}{v(x)}\,dx\right) = S_0
```

この条件は、合計つらさに関する制約として、次のように書き換えることもできる。

```math
\left(\int_0^L \frac{(a v(x)^3 + b(G(x))v(x))^4}{v(x)}\,dx\right) - \left(S_0\int_0^L \frac{1}{v(x)}\,dx\right) = 0
```

積分区間が同じであるからまとめてしまってよい：

```math
\int_0^L \left(\frac{(a v(x)^3 + b(G(x))v(x))^4}{v(x)} - \frac{S_0}{v(x)}\right)dx = 0
```

ラグランジュの未定乗数法を用いる。次のような関数を考える。

```math
F(v, \lambda) := \int_0^L \left(\frac{1}{v(x)} + \lambda\left(\frac{(a v(x)^3 + b(G(x))v(x))^4}{v(x)} - \frac{S_0}{v(x)}\right)\right)dx
```

この最適化問題の解は、上式の $`v, \lambda`$ に関する極値である。ひとまず $`v`$ での偏微分について考えてみよう。

各点での速度は独立に考えてしまってよさそうだから、被積分関数について考える（$`b(G(x))`$ は単に $`b`$、$`v(x)`$ は単に $`v`$ と書いてしまうことにする）。

```math
f(v, \lambda) := \frac{1}{v} + \lambda\left(\frac{(av^3 + bv)^4}{v} - \frac{S_0}{v}\right)
= \lambda(av^2 + b)^4 v^3 + \frac{1 - \lambda S_0}{v}
```

この関数の $`v`$ に関する偏微分は、

```math
\frac{\partial f}{\partial v} = \lambda(11av^2 + 3b)(av^2 + b)^3 v^2 - \frac{1 - \lambda S_0}{v^2}
```

となって、これが0というのは、$`v`$ に関する方程式

```math
(11av^2 + 3b)(av^2 + b)^3 v^4 = \frac{1}{\lambda} - S_0
```

が満たされるということである。

#### 戦略的な意味

上式の持つ意味について考えたい。三次抵抗係数が $`a`$、一次抵抗係数が $`b`$、いまの走行速度が $`v`$ だったとして、微小区間の走行時間を少し変えるとつらさはどう変わるだろうか？これは、$`t := 1/v`$ とおいて、$`S := (av^3 + bv)^4 / v`$ の $`t`$ に関する偏微分 $`\delta S/\delta t`$ を考えることにほかならない。計算すると、$`\delta S/\delta t = -(11av^2 + 3b)(av^2 + b)^3 v^4`$ となり、これは上式左辺の-1倍である。

つまり、$`(11av^2 + 3b)(av^2 + b)^3 v^4 = 1/\lambda - S_0`$ というのは、$`\delta S/\delta t = S_0 - 1/\lambda`$ という意味であり、端的に言えば「つらさを増やしてタイム短縮を購入するとき、どの地点も同じ値段 $`S_0 - 1/\lambda`$ になるようにする」という戦略である。

#### 最適速度

さて、ここまでの議論ではコース C と力量 S_0 の二つがなければ λ が定まらなかったが、上述の結論を $`\delta S/\delta t = \mathrm{const.}`$ だと思うと、すべてのコースに共通して適用できる結果が得られる。

平均つらさのかわりに、勾配0の区間で発揮する予定のパワー $`P_0`$ を与えて、その上での最適なペーシングを考えてみよう。このような問題設定の変更は合理的である。たのしくロングライドするのが目的であれば、コースの勾配プロファイルによらず平坦路も登りも「いつもどおり」のペースで走るのが自然であろう。

**Def. 11** 抵抗モデル $`M (= (a, b))`$ 、「平地パワー」正実数 $`P_0`$ と、$`\theta\in[-\pi/2, \pi/2]`$ に対し、「最適速度 $`v_{\mathrm{op}}(M, P_0, \theta)`$」を次のように定める：

- $`v_0`$ を、$`av_0^3 + b(0)v_0 = P_0`$ を満たす正の値として定める。
- $`c := (11av_0^2 + 3b(0))(av_0^2 + b(0))^3 v_0^4`$ とおく。（これを「最適コスト」と呼ぶことにする）
- $`v_{\mathrm{op}}(M, P_0, \theta)`$ を、$`(11av^2 + 3b(\theta))(av^2 + b(\theta))^3 v^4 = c`$ を満たす最大の $`v`$ とする。（この方程式はかならず均衡パワーが正になるような正速度の解を持つ。$`b(G(x))<0`$ のとき正の解を複数持つが、最大以外の解は均衡パワーが負の値になる速度であり、負の仕事をしても脚が楽になることはない。）

適当なコース $`C`$ について、走行 $`R (= (C, M, \text{各点 } x \text{ に対し } v_{\mathrm{op}}(M, P_0, G(x))))`$ を考えると、先の議論から、これは $`R`$ と同じ平均つらさを持つ $`C`$ の走行の中で最速のものである。

最適速度と同様に、最適パワー、最適つらさについても定義しておこう。これらの定性的な振る舞いは次節で議論する。

**Def. 12** 抵抗モデル $`M (= (a, b))`$、「平地パワー」正実数 $`P_0`$ と、$`\theta\in[-\pi/2, \pi/2]`$ に対し、「最適パワー $`P_{\mathrm{op}}(M, P_0, \theta)`$」を、$`P_{\mathrm{op}}(M, P_0, \theta) := av_{\mathrm{op}}(M, P_0, \theta)^3 + b(\theta)v_{\mathrm{op}}(M, P_0, \theta)`$ と定める。また、「最適つらさ $`S_{\mathrm{op}}(M, P_0, \theta)`$」を、$`S_{\mathrm{op}}(M, P_0, \theta) := P_{\mathrm{op}}(M, P_0, \theta)^4 / v_{\mathrm{op}}(M, P_0, \theta)`$ と定める。なお、上述のとおり、最適パワーはかならず正である。

かくして、我々は「平地をある特定のパワーで走る」という仮定のもと、勾配ごとに最適な速度・パワーが定まること、またそれがコースによらないことを示した。

### 最適なペーシングの性質

#### 速度・パワー・つらさの概形

最適なペーシングでは、勾配に対して速度・パワー・区間つらさがどう変化するか調べてみよう。最適コストを $`c`$ とする。

抵抗のうち一次の係数の内実は、$`b(\theta) = mg(\sin\theta + C_{rr}\cos\theta)/\eta`$ であったから、おおむね $`b(\theta) \propto \theta`$ である。最適コストと速度の関係は $`(11av^2 + 3b)(av^2 + b)^3 v^4 = c`$ であったから、最適速度は勾配に対して単調減少である。

ある区間の最適速度が $`v`$、パワーが $`P`$ だったとして、コストの条件を変形すると $`c = P^3(3P + 8av^3)`$ となるから、$`v`$ と $`P`$ はトレードオフの関係にあることがわかる。したがって、パワーは勾配に対して単調増加である。速度が単調減少でパワーが単調増加なのだから、区間つらさが勾配に対して単調増加であることもわかる。

#### 急坂路での概形

空気抵抗が無視できる急坂路では、増減以上に強い結果が得られる。$`\theta`$ が大きいと $`v_{\mathrm{op}}(M, P_0, \theta)`$ が小さくなるため、速度の二乗の項は無視して近似できる。

この領域では、コストと速度の関係は $`c \approx 3P^4 \approx 3(vb(\theta))^4`$ であるから、最適速度は $`v_{\mathrm{op}}(M, P_0, \theta) \approx (c/3)^{1/4} / b(\theta) \propto 1/\theta`$ である。

さらに、平地パワー $`P_0`$ と $`v_{\mathrm{op}}(M, P_0, \theta)`$ の比を考えてみよう。
$`b(0)`$ を 0 とみなすと、$`P_0 \approx av_0^3`$ であるから、最適コストは $`c \approx 11P_0^4`$ である。

一方、$`P_{\mathrm{op}}(M, P_0, \theta) \approx b(\theta)v_{\mathrm{op}}(M, P_0, \theta)`$ であるから、これらを合わせて $`3P_{\mathrm{op}}(M, P_0, \theta)^4 \approx 11P_0^4`$ となる。つまり、急坂路での最適パワーは平地パワーの $`(11/3)^{1/4} \approx 1.38`$ 倍程度になることがわかる。（実際には $`b(0)`$ は0でないので、倍率はこれより少し低くなる。

上述の議論より、急坂路において速度は斜度に反比例し、パワーはおおよそ一定であるから、区間つらさは勾配に対して一次関数に漸近することになる。

平地と急坂路の区間つらさの比を考える。平地の区間つらさはおよそ $`P_0^4 / v_0 \approx P_0^4 / (P_0/a)^{1/3} = a^{1/3} P_0^{11/3}`$ である。一方、急坂路の区間つらさはおよそ $`(11/3\,P_0^4) / v_{\mathrm{op}}(M, P_0, \theta) \approx (11/3\,P_0^4) / ((11/3)^{1/4} P_0 / \theta) = (11/3)^{3/4}\theta P_0^3`$ である。
したがって、急坂路の区間つらさは平地の区間つらさの $`(11/3)^{1/4} a^{-1/3} P_0^{-2/3}\theta`$ 倍程度になることがわかる。この結果の特筆すべき点は三つである：

- 急坂路において、距離あたりのつらさはおおむね勾配に比例する。
- 比例係数は $`a^{1/3}`$ に反比例する。空気抵抗係数が大きいと平地がつらくなるため、空気抵抗の影響が少ない急坂路の相対的なつらさは小さくなる。
- 比例係数は $`P_0^{-2/3}`$ に反比例する。パワーが大きいとペースがあがるが、つらさの増大は急坂路より平地のほうが大きい。

ここから「$`P_0`$ の小さな初心者ほど相対的に坂をつらく感じる」という主張が得られるが、これはサイクリストの実感とよく符合する。

#### 最適速度の数値解について

$`v_{\mathrm{op}}(M, P_0, \theta)`$ に関する方程式の解を数値的に求める際の考察として、コストと速度の関係を抜き出した関数 $`f(v) := (11av^2 + 3b)(av^2 + b)^3 v^4`$ の振る舞いを調べておこう。

- $`g(V) := (11aV + 3b)(aV + b)^3 V^2`$ とおくと、$`f(v) = g(v^2)`$ である。我々は正の速度にしか興味がないので、求解にあたっては $`g(V) = c`$ を解き、その平方根を取ればよい。
- $`g'(V) = 6V (aV + b)^2 (11a^2V^2 + 8abV + b^2)`$ である。
- $`b`$ の正負で解の挙動が変わる。
  - $`b`$ が正ならば、$`V>0`$ で $`g`$ は単調増加である。$`v_0`$ などから出発してニュートン法で解を求めるなどすればよい。
  - $`b`$ が負ならば、$`V=0`$, $`-((4\pm\sqrt{5})/11)\,b/a`$（それぞれ $`-0.16b/a`$, $`-0.56b/a`$ 程度）, $`-b/a`$ で $`g'(V)=0`$ となる。我々が興味のある $`V`$ は $`-b/a`$ より大きいので、$`-2b/a`$ などから出発してニュートン法で解を求めるなどすればよい。

## 指標の提案

### 平地換算距離

ここまでで得られた最適ペーシングにおける各区間のつらさを用いて、サイクリングコースの大変さを表す指標「平地換算距離」（NFD: Normalized Flat Distance）を提案する。

**Def. 13** 抵抗モデル $`M`$、平地パワー $`P_0`$、正実数 $`v_{\min}`$ と、斜度 $`\theta\in[-\pi/2, \pi/2]`$ に対して、

- 「想定速度 $`v_{\mathrm{est}}(M, P_0, \theta)`$」を、$`\max(v_{\mathrm{op}}(M, P_0, \theta), v_{\min})`$ と定める。
- 「想定パワー $`P_{\mathrm{est}}(M, P_0, \theta)`$」を、$`a v_{\mathrm{est}}(M, P_0, \theta)^3 + b(\theta)v_{\mathrm{est}}(M, P_0, \theta)`$ と定める。
- 「想定区間つらさ $`S_{\mathrm{est}}(M, P_0, \theta)`$」を、$`P_{\mathrm{est}}(M, P_0, \theta)^4 / v_{\mathrm{est}}(M, P_0, \theta)`$ と定める。
- 「勾配係数 $`k(M, P_0, \theta)`$」を、$`S_{\mathrm{est}}(M, P_0, \theta) / S_{\mathrm{est}}(M, P_0, 0)`$ と定める。

ここで、勾配係数というのは「斜度θの1kmは平地1kmの何倍つらいか」である。

**Def. 14** 抵抗モデル $`M`$、平地パワー $`P_0`$、正実数 $`v_{\min}`$ とコース $`C (= (L, G))`$ に対して、「平地換算距離 $`D_{\mathrm{eq}}(M, P_0, v_{\min}, C)`$」を、$`\int_0^L k(M, P_0, G(x))\,dx`$ と定める。

速度の下限を設けたのは、筋肉の性質上効率的なペダリングができる回転数に下限があるためである。典型的なロードバイクの最低ギアはクランク1周あたり2m程度であるが、これを分速40回転強で回せば5km/hである。これを下回るとまともに坂を登れないため、$`D_{\mathrm{eq}}`$ の計算においては無理してでも $`v_{\min}`$ を出すものとしてつらさを計算した。

### 計算手順

平地換算距離の計算手順をあらためてまとめる。

まず、抵抗モデル $`M`$（三次の係数 $`a`$、勾配から一次の係数を得る $`b`$）と平地パワー $`P_0`$ に対して、勾配係数 $`k(M, P_0, \theta)`$ を計算する。

1. 「平地速度」 $`v_0`$ は方程式 $`av_0^3 + b(0)v_0 = P_0`$ の唯一の実数解であるから求める。（$`a, b(0)`$ はいずれも正であるから、これを満たす $`v_0`$ は一意に存在し、正である。）
2. 「最適コスト」 $`c`$ を、$`c := (11av_0^2 + 3b(0))(av_0^2 + b(0))^3 v_0^4`$ とおく。
3. 斜度 $`\theta`$ における最適速度 $`v_{\mathrm{op}}(M, P_0, \theta)`$、すなわち $`v`$ の方程式 $`(11av^2 + 3b(\theta))(av^2 + b(\theta))^3 v^4 = c`$ の最大の実数解を求める。（先に述べた通り、この方程式は常に解を持ち、最大の実数解ではパワーが常に正である。）
4. 斜度 $`\theta`$ における想定速度 $`v_{\mathrm{est}}(M, P_0, \theta)`$ を、$`\max(v_{\mathrm{op}}(M, P_0, \theta), v_{\min})`$ を計算する。
5. 斜度 $`\theta`$ における想定パワー $`P_{\mathrm{est}}(M, P_0, \theta)`$ を、$`av_{\mathrm{est}}(M, P_0, \theta)^3 + b(\theta)v_{\mathrm{est}}(M, P_0, \theta)`$ を計算する。
6. 斜度 $`\theta`$ における想定区間つらさ $`S_{\mathrm{est}}(M, P_0, \theta)`$ を、$`P_{\mathrm{est}}(M, P_0, \theta)^4 / v_{\mathrm{est}}(M, P_0, \theta)`$ を計算する。
7. 斜度 $`\theta`$ における勾配係数 $`k(M, P_0, \theta)`$ を、$`S_{\mathrm{est}}(M, P_0, \theta) / S_{\mathrm{est}}(M, P_0, 0)`$ を計算する。

次に、与えられたコース $`C`$（コース長 $`L`$、勾配関数 $`G`$）について、勾配係数を用いて平地換算距離を計算する。

8. コース $`C`$ の平地換算距離 $`D_{\mathrm{eq}}(M, P_0, v_{\min}, C)`$ を、$`\int_0^L k(M, P_0, G(x))\,dx`$ を計算する。

実用上の留意点として、NFDの計算を行う前に勾配データの平滑化を行うことが推奨される。勾配係数は想定速度が最低速度を下回るような勾配（$`P_0=100\,\mathrm{W}`$ であれば、おおよそ13%以上）では急激に大きな値を取る。既存のルート作成サービスでは標高データを直接用いて計算しているため、エクスポートしたデータには道路の位置ズレや誤差に影響されて実態と乖離した急傾斜が含まれることがある。次節では、サビツキーフィルタを用いて平滑化を行っている。

## 結論

### 今後の課題

今回提案した手法では、出力を均衡パワーに、運動の負荷をその四乗に単純化して考えることで、サイクリングに関する一般的な諸結論と同様の結果や加法的なコース難易度指標を得た。しかしながら、ここまで挙げた手法では、集団走行による空気抵抗の低減、酸素濃度の違いに起因するパワーの変化などは考慮しておらず、乳酸濃度の法則である Cogan の四乗則をそのまま用いることの妥当性も評価していない。より細やかなパワー・運動負荷のモデルに対して同様の手法を適用することで、さらに実戦的な指標を得ることができると考えられる。

### まとめ

本稿では、Cogan の四乗則に基づき、サイクリングを一種の最適化問題として定式化し、一般的。また、定式化の結果を用いて、サイクリングコースの難易度を表す指標「平地換算距離」を提案した。
