## 序論
ハイキング情報共有サイト YAMAP では、コースの大変さを「コース定数」という加法的な指標で表し、合計12未満ならやさしいコース、25以上ならきつめのコースであるというふうに目安を示して、登山計画作成の一助としている。

サイクリングコースの難易度は距離や登った高さの合計（獲得標高）によって間接的に表されるが、これらを一次元の指標にまとめて比較することはできないだろうか。たとえば、次の四つのコースについて考えてみる。
- 平坦100km
- 2%の登り10km
- 4%の登り5km
- 20%の登り1km
獲得標高という点において2% 10km、4% 5km、20% 1km はすべて同じ 200m up であるが、4% 5km は素人に登りきれても 20% 1km ははっきり言って無理難題である（目安として、坂の舗装がコンクリになって丸い滑り止めが付くのが15%程度である）。また、ふつう2% 10km よりは平坦100kmのほうが大変であるが、距離と高さを統一的に扱う手段がなければ、これらの大変さをうまく比べることができない。

ところで、スポーツ科学の分野では、サイクリストが自転車を走らせるために投入する仕事率、一般には「パワー」と呼ばれているものであるが、これを測定して瞬間的な運動負荷を定量的に扱う方法が普及している。走行に必要なパワーは速度と勾配から決まり、たとえば「このサイクリストが平坦路を31km/hで走るのと、4%の坂を19km/hで走るのはいずれも200W程度必要です」というようなことがいえ、平坦路と坂道の瞬間的な走行負荷をパワーという一つの指標で表すことができる。コースをどのようなペースで走るかさえ決めてしまえば、各点でどのくらいのパワーが必要になるか計算することができるから、これをもとにコース全体の大変さを議論すればよいだろう。

さしあたって難しい点は、平坦路と急坂路で速度に対するパワーの増え方が異なることである。次節で触れる通り、パワーの表式のうち、平坦路においては空気抵抗に起因する速度の三乗の項が、急坂路では位置エネルギー変化に起因する速度の一次の項が支配的である。パワー一定でサイクリングをすると空気抵抗に法外なパワーを支払うことになるから、もうすこし賢いペーシングを考えねばならない。

本稿では、サイクリングコースを走破する「つらさ」をスポーツ科学的側面から考察して定義し、おなじつらさで早く走れる「賢いペーシング」を最適化問題として定式化する。この問題の解からは、サイクルスポーツで広く知られている事実が再発見される：
- 初心者ほど登りをつらく感じる。
- 平坦路ではほどほどに手を抜き、登りでは頑張ったほうがよい。
- まともに登れる坂であれば、獲得標高はつらさのよい指標である。
そののち、得られたペーシングに現実的な修正を加えて、サイクリングコースの大変さを表す新しい指標「平地換算距離」を提案する。

## 物理的・生理学的前提
### パワーのモデル
サイクリングの客観的な運動負荷は、サイクリストがクランクに対して発揮する仕事率（「パワー」と呼ばれる）によって測定される。無風の条件下で角度θの道路を速度vで定速走行する場合のパワー P_precise(θ, v) は、次の式で表される：
`P_precise(θ, v) = (1/2*ρCdA * v^3 + mg(sin(θ) + Crr*cos(θ)) * v + m * v * dv/dt)/η`
ここで、ρは空気の密度、Cdは空気抵抗係数、Aは前面投影面積、mは総質量、gは重力加速度、Crrは転がり抵抗係数、ηは駆動効率である。これらの係数は、サイクリングの一般的な条件下ではおおむね定数であるとみなすことができる。
これらの大まかな値を示すと、気温20℃、海面上の空気の密度はおよそ1.2 kg/m^3である。ロードバイクの場合、CdAはおよそ0.4 m^2、Crrはおよそ0.005、ηは95~98%である。また、車重は6〜10kg、サイクリストの体重は50〜100kg程度であり、100~300W程度のパワーを継続して発揮できる。

今回は指標を単純にするため、加減速に由来するパワーを無視する（サイクリストがクランクにする仕事のほとんどは位置エネルギーを得たり風を起こしたりするために空費され、運動エネルギーのやりとりは大きくない。全質量70kg、速度10m/sのときの運動エネルギーはたったの3500Jであるが、これは位置エネルギー5m分にしかならないし、この速度域での空気抵抗の仕事率は250Wに達する）。
また、上記の係数は、一回のサイクリングで大きく変わることはないから定数であるとみなすことにする。これ以降、係数を a∈ℝ と、b : [-π/2, π/2] → ℝ つまり、
a := 1/2*ρCdA/η
b(θ) := mg(sin(θ) + Crr*cos(θ))/η
にまとめて、P_precise の近似値 P_approx(θ, v) := a * v^3 + b(θ) * v を考えることにする。これは速度を均衡させるために必要なパワーであるから、「均衡パワー」と呼ぶことにする。

### 主観的な運動負荷のモデル
Andrew Coggan らは、サイクリストの発揮したパワーと筋肉中の乳酸濃度(後者がおよそ前者の4乗に比例するらしい)の関係から、Normalized Power® (NP)という指標を提案し、Stravaを始めとしたアクティビティ記録サービスで採用されている。NPは、その回の運動で発揮したパワーを30秒移動平均で平滑化したのちL4ノルムを取ったものである。サイクリングを頑張った際のNPはサイクリストの力量によっておおまかに決まることが知られている。

NPの四乗は加法的な指標となる。ここでは、コースをある定められた速度分布のもと走った際の均衡パワー^4の時間積分（「合計つらさ」と呼ぶことにする）をサイクリストの主観的な運動負荷の指標とする。

### 走行の最適化
#### おきもち
我々サイクリストが達成できるNPは力量によって決まっているから、当然サイクリングにおいては同じNPで最も早くコースを走り切りたいと考える。（「別に急いでないし」と思ったそこのあなたは、「同じタイムで最も低いNP」だと考えてもよい。これはゴニョゴニョすると同種の問題に帰着する）
本節では、前節で定義した「平均つらさ」を一定に保ちつつ、走行時間を最小化する最適な走行がどのようものであるか考えることにする。定性的には、次のような結論が得られる。
- 各地点で発揮すべきパワーは、その地点の勾配のみによって定まる。
- 平地ではパワーを少なめに、登りではパワーを多めに発揮するべきである。（平地で飛ばしても空気抵抗が増えるばかりで割に合わないため、このような配分になる。この結果は我々の実感と符合する。）
- 勾配-つらさのプロットは登り勾配側では一次関数に漸近する。
- 急坂路でのパワーは抵抗モデルによらず平地の1.3~1.38倍程度になる。
- 驚くべきことに、抵抗モデルと特定の勾配での速度をひとつ定めると、どのようなコースであっても同じペーシングが最適となる。
特に、パワーに関する諸結論はサイクリングの一般的アドバイスと整合的であるから、次節が与える結論は荒唐無稽ではないだろう。

#### 定式化
特定の抵抗モデル M (= (a, b)) にとって各コースで最適である走行がどのようなものか調べてみよう。

仮の問題設定として、サイクリストの力量を表す正実数 S_0 とコース C (= (L, G))が与えられたとき、v: [0, L]→ℝ_+ であって、走行 (C, M, v) の平均つらさが S_0 であり、走行時間が最小となるものを考えることにする。

この走行の合計つらさは、各点xにおける地点速度をv(x)として、∫_0^L (a * v(x)^3 + b(G(x)) * v(x))^4 dx である。また、走行時間は ∫_0^L 1/v(x) dx である。これらを用いて、平均つらさが S_0 であることとは、次の等式が満たされることである。
`(∫_0^L (av(x)^3 + b(G(x))v(x))^4 / v(x) dx) / (∫_0^L 1/v(x) dx) = S_0`

この条件は、合計つらさに関する制約として、次のように書き換えることもできる。
`(∫_0^L (av(x)^3 + b(G(x))v(x))^4 / v(x) dx) - (S_0 ∫_0^L 1/v(x) dx) = 0`

積分区間が同じであるからまとめてしまってよい：
`∫_0^L (av(x)^3 + b(G(x))v(x))^4 / v(x) - S_0 / v(x) dx = 0`

ラグランジュの未定乗数法を用いる。次のような関数を考える。
`F(v, λ) := ∫_0^L 1/v(x) + λ((av(x)^3 + b(G(x))v(x))^4 / v(x) - S_0 / v(x)) dx`

この最適化問題の解は、上式のv, λに関する極値である。ひとまず v での偏微分について考えてみよう。

各点での速度は独立に考えてしまってよさそうだから、被積分関数について考える（b(G(x))は単にb、v(x)は単にvと書いてしまうことにする）。
`f(v, λ) := 1/v + λ((av^3 + bv)^4 / v - S_0 / v)`
`= λ(av^2 + b)^4 v^3 + (1 - λS_0) / v`

この関数のvに関する偏微分は、
`∂f/∂v = λ(11av^2 + 3b)(av^2 + b)^3 v^2 - (1 - λS_0) / v^2`
となって、これが0というのは、vに関する方程式
`(11av^2 + 3b)(av^2 + b)^3 v^4 = 1/λ - S_0`
が満たされるということである。

#### 戦略的な意味
上式の持つ意味について考えたい。三次抵抗係数がa、一次抵抗係数がb、いまの走行速度がvだったとして、微小区間の走行時間を少し変えるとつらさはどう変わるだろうか？これは、`t:=1/v`とおいて、`S := (av^3 + bv)^4 / v` の t に関する偏微分 δS/δt を考えることにほかならない。計算すると、`δS/δt = -(11av^2 + 3b)(av^2 + b)^3 v^4` となり、これは上式左辺の-1倍である。

つまり、`(11av^2 + 3b)(av^2 + b)^3 v^4 = 1/λ - S_0` というのは、`δS/δt = S_0 - 1/λ` という意味であり、端的に言えば「つらさを増やしてタイム短縮を購入するとき、どの地点も同じ値段`S_0 - 1/λ`になるようにする」という戦略である。

#### 最適速度
さて、ここまでの議論ではコース C と力量 S_0 の二つがなければ λ が定まらなかったが、上述の結論を `δS/δt = const.` だと思うと、すべてのコースに共通して適用できる結果が得られる。

平均つらさのかわりに、勾配0の区間で発揮する予定のパワー P_0 を与えて、その上での最適なペーシングを考えてみよう。このような問題設定の変更は合理的である。たのしくロングライドするのが目的であれば、コースの勾配プロファイルによらず平坦路も登りも「いつもどおり」のペースで走るのが自然であろう。

def. 11 抵抗モデル M (= (a, b)) 、「平地パワー」正実数 P_0 と、θ∈[-π/2, π/2] に対し、「最適速度 v_op(M, P_0, θ)」を次のように定める：
- `v_0` を、`av_0^3 + b(0)v_0 = P_0` を満たす正の値として定める。
- `c := (11av_0^2 + 3b(0))(av_0^2 + b(0))^3 v_0^4` とおく。（これを「最適コスト」と呼ぶことにする）
- v_op(M, P_0, θ) を、`(11av^2 + 3b(θ))(av^2 + b(θ))^3 v^4 = c` を満たす最大のvとする。（この方程式はかならず均衡パワーが正になるような正速度の解を持つ。 b(G(x))<0 のとき正の解を複数持つが、最大以外の解は均衡パワーが負の値になる速度であり、負の仕事をしても脚が楽になることはない。）

適当なコース C について、走行 R (= (C, M, 各点 x に対し v_op(M, P_0, G(x)))) を考えると、先の議論から、これは R と同じ平均つらさを持つ C の走行の中で最速のものである。

最適速度と同様に、最適パワー、最適つらさについても定義しておこう。これらの定性的な振る舞いは次節で議論する。

def. 12 抵抗モデル M (= (a, b)) 、「平地パワー」正実数 P_0 と、θ∈[-π/2, π/2] に対し、「最適パワー P_op(M, P_0, θ)」を、`P_op(M, P_0, θ) := av_op(M, P_0, θ)^3 + b(θ)v_op(M, P_0, θ)` と定める。また、「最適つらさ S_op(M, P_0, θ)」を、`S_op(M, P_0, θ) := P_op(M, P_0, θ)^4 / v_op(M, P_0, θ)` と定める。なお、上述のとおり、最適パワーはかならず正である。

かくして、我々は「平地をある特定のパワーで走る」という仮定のもと、勾配ごとに最適な速度・パワーが定まること、またそれがコースによらないことを示した。

### 最適なペーシングの性質
#### 速度・パワー・つらさの概形
最適なペーシングでは、勾配に対して速度・パワー・区間つらさがどう変化するか調べてみよう。最適コストを c とする。

抵抗のうち一次の係数の内実は、`b(θ) = mg(sin(θ) + Crr*cos(θ))/η` であったから、おおむね `b(θ) ∝ θ` である。最適コストと速度の関係は `(11av^2 + 3b)(av^2 + b)^3 v^4 = c` であったから、最適速度は勾配に対して単調減少である。

ある区間の最適速度がv、パワーがPだったとして、コストの条件を変形すると `c = P^3 (3P + 8av^3)` となるから、v と P はトレードオフの関係にあることがわかる。したがって、パワーは勾配に対して単調増加である。速度が単調減少でパワーが単調増加なのだから、区間つらさが勾配に対して単調増加であることもわかる。

#### 急坂路での概形
空気抵抗が無視できる急坂路では、増減以上に強い結果が得られる。θが大きいと v_op(M, P_0, θ) が小さくなるため、速度の二乗の項は無視して近似できる。

この領域では、コストと速度の関係は `c ≈ 3P^4 ≈ 3(vb(θ))^4` であるから、最適速度は `v_op(M, P_0, θ) ≈ (c/3)^(1/4) / b(θ) ∝ 1/θ` である。

さらに、平地パワー P_0 と v_op(M, P_0, θ) の比を考えてみよう。
b(0) を 0 とみなすと、`P_0 ≈ av_0^3` であるから、最適コストは `c ≈ 11P_0^4` である。

一方、`P_op(M, P_0, θ) ≈ b(θ)v_op(M, P_0, θ)` であるから、これらを合わせて `3P_op(M, P_0, θ)^4 ≈ 11P_0^4` となる。つまり、急坂路での最適パワーは平地パワーの `(11/3)^(1/4) ≈ 1.38` 倍程度になることがわかる。（実際には b(0) は0でないので、倍率はこれより少し低くなる。

上述の議論より、急坂路において速度は斜度に反比例し、パワーはおおよそ一定であるから、区間つらさは勾配に対して一次関数に漸近することになる。

平地と急坂路の区間つらさの比を考える。平地の区間つらさはおよそ `P_0^4 / v_0 ≈ P_0^4 / (P_0/a)^(1/3) = a^(1/3) * P_0^(11/3)` である。一方、急坂路の区間つらさはおよそ `(11/3 P_0^4) / v_op(M, P_0, θ) ≈ (11/3 P_0^4) / ((11/3)^(1/4) P_0 / θ) = (11/3)^(3/4) * θP_0^3` である。
したがって、急坂路の区間つらさは平地の区間つらさの `(11/3)^(1/4) a^(-1/3) P_0^(-2/3) θ` 倍程度になることがわかる。この結果の特筆すべき点は三つである：
- 急坂路において、距離あたりのつらさはおおむね勾配に比例する。
- 比例係数は a^(1/3) に反比例する。空気抵抗係数が大きいと平地がつらくなるため、空気抵抗の影響が少ない急坂路の相対的なつらさは小さくなる。
- 比例係数は P_0^(-2/3) に反比例する。パワーが大きいとペースがあがるが、つらさの増大は急坂路より平地のほうが大きい。

ここから「P_0 の小さな初心者ほど相対的に坂をつらく感じる」という主張が得られるが、これはサイクリストの実感とよく符合する。

#### 最適速度の数値解について
v_op(M, P_0, θ) に関する方程式の解を数値的に求める際の考察として、コストと速度の関係を抜き出した関数 `f(v) := (11av^2 + 3b)(av^2 + b)^3 v^4` の振る舞いを調べておこう。
- `g(V) := (11aV + 3b)(aV + b)^3 V^2` とおくと、`f(v) = g(v^2)` である。我々は正の速度にしか興味がないので、求解にあたっては `g(V) = c` を解き、その平方根を取ればよい。
- `g'(V) = 6V (aV + b)^2 (11a^2V^2 + 8abV + b^2)` である。
- b の正負で解の挙動が変わる。
  - b が正ならば、V>0 で g は単調増加である。v_0 などから出発してニュートン法で解を求めるなどすればよい。
  - b が負ならば、V=0, -((4±√5)/11) * b/a（それぞれ -0.16b/a, -0.56b/a 程度）, - b/a で g'(V) = 0 となる。我々が興味のある V は -b/a より大きいので、-2b/a などから出発してニュートン法で解を求めるなどすればよい。

## 指標の提案
### 平地換算距離
ここまでで得られた最適ペーシングにおける各区間のつらさを用いて、サイクリングコースの大変さを表す指標「平地換算距離」（NFD: Normalized Flat Distance）を提案する。

def. 13 抵抗モデル M、平地パワー P_0 、正実数 v_min と、斜度 θ∈[-π/2, π/2] に対して、
- 「想定速度 v_est(M, P_0, θ)」を、max(v_op(M, P_0, θ), v_min) と定める。
- 「想定パワー P_est(M, P_0, θ)」を、`av_est(M, P_0, θ)^3 + b(θ)v_est(M, P_0, θ)`
- 「想定区間つらさ S_est(M, P_0, θ)」を、`P_est(M, P_0, θ)^4 / v_est(M, P_0, θ)` と定める。
- 「勾配係数 k(M, P_0, θ)」を、`S_est(M, P_0, θ) / S_est(M, P_0, 0)` と定める。

ここで、勾配係数というのは「斜度θの1kmは平地1kmの何倍つらいか」である。

def. 14 抵抗モデル M、平地パワー P_0 、正実数 v_min とコース C (= (L, G)) に対して、「平地換算距離 D_eq(M, P_0, v_min, C)」を、`∫_0^L k(M, P_0, G(x)) dx` と定める。

速度の下限を設けたのは、筋肉の性質上効率的なペダリングができる回転数に下限があるためである。典型的なロードバイクの最低ギアはクランク1周あたり2m程度であるが、これを分速40回転強で回せば5km/hである。これを下回るとまともに坂を登れないため、D_eq の計算においては無理してでも v_min を出すものとしてつらさを計算した。

### 計算手順
平地換算距離の計算手順をあらためてまとめる。

まず、抵抗モデル M (= (三次の係数 a, 勾配から一次の係数を得る b)) と平地パワー P_0 に対して、勾配係数 k(M, P_0, θ) を計算する。
1. 「平地速度」 v_0 を、`av_0^3 + b(0)v_0 = P_0` を満たす値として定める。（a, b(0)はいずれも正であるから、これを満たすv_0は一意に存在し、正である。）
2. 「最適コスト」 c を、`c := (11av_0^2 + 3b(0))(av_0^2 + b(0))^3 v_0^4` とおく。
3. 斜度 θ における最適速度 v_op(M, P_0, θ) を、`(11av^2 + 3b(θ))(av^2 + b(θ))^3 v^4 = c` を満たす最大のvとして定める。（先に述べた通り、この方程式は常に解を持ち、最大の実数解ではパワーが常に正である。）
4. 斜度 θ における想定速度 v_est(M, P_0, θ) を、max(v_op(M, P_0, θ), v_min) と定める。
5. 斜度 θ における想定パワー P_est(M, P_0, θ) を、`av_est(M, P_0, θ)^3 + b(θ)v_est(M, P_0, θ)` と定める。
6. 斜度 θ における想定区間つらさ S_est(M, P_0, θ) を、`P_est(M, P_0, θ)^4 / v_est(M, P_0, θ)` と定める。
7. 斜度 θ における勾配係数 k(M, P_0, θ) を、`S_est(M, P_0, θ) / S_est(M, P_0, 0)` と定める。

次に、与えられたコース C (=(コース長 L, 勾配関数 G)) について、勾配係数を用いて平地換算距離を計算する。
8. コース C の平地換算距離 D_eq(M, P_0, v_min, C) を、`∫_0^L k(M, P_0, G(x)) dx` と定める。
